# XMU-ChemLearning

整理了在 xmu 化院的学习资料，如果有侵权内容请联系我删除

希望对你有所帮助，也欢迎提交 PR 补充资料

## 使用说明

由于 `.git` 目录较大，在 `clone` 时建议使用 `--depth 1` 来跳过对之前版本的追踪，节省时间和空间。
也可以直接下载 zip 或者下载需要的单文件。
