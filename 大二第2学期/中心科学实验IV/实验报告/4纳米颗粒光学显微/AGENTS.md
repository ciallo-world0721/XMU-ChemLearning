# AGENTS.md

## 项目目标
- 本项目用于生成“纳米颗粒光学显微成像”实验报告。
- 最终报告源文件应为项目根目录下的 `report.tex`，最终 PDF 为 `report.pdf`。

## 报告格式约束
- 严格沿用 `template/report.tex` 的页面、页眉页脚、字体和基础宏定义。
- 不要直接修改 `template/` 目录；如需编辑报告，应修改根目录 `report.tex`。
- 页眉中的系、专业、学号、姓名、日期、实验台号等空栏元素应保留，不要删除。
- 使用 `xelatex` 编译。

## 当前内容取舍
- 轨迹数据展示第 `1`、`2`、`3-2`、`4-2` 组 MATLAB 输出图；其中只需要详细讨论第 `3-2` 组。
- 不在报告中展示第 `3` 组和第 `4` 组轨迹数据。
- 报告中 MSD 曲线使用 MATLAB 已输出的图 `src/img/Figure_1.png`、`src/img/Figure_2.png`、`src/img/Figure_3-2.png`、`src/img/Figure_4-2.png`；不要再用 Python 重新绘制这些 MSD 图。
- 可保留 Python/matplotlib 绘制的扩散参数柱状图，但应仅比较报告展示的代表性组别。
- “思考题”应作为第 4 部分，“结论”应作为第 5 部分。
- 思考题题目与 `legacy/` 中旧报告保持一致，但答案需要重新组织和生成，避免照抄。

## 图片与数据
- 新增或引用图片优先放在 `src/img/`。
- 中文文件名图片如需在 LaTeX 中稳定引用，可保留英文副本。
- 已有 MATLAB 输出图包括 `src/img/Figure_*.png`；除非用户明确要求，不要删除已有数据处理或绘图脚本。
