\section{引言}
\vspace{-0.5em}

组蛋白 H3 是核小体的核心组分，其 N 端尾部可发生乙酰化、甲基化和磷酸化等多种翻译后修饰，
在基因转录调控、DNA 复制与损伤修复中具有重要作用。为研究特定位点修饰的结构与功能，
需要获得序列明确、结构均一且可定点改造的 H3 蛋白。与天然提取和常规重组表达相比，蛋白质化学合成策略在位点可控性方面更具优势，
其中天然化学连接（Native Chemical Ligation, NCL）是最具代表性的技术路线之一\cite{dawson1994,kent2009}。

NCL 的核心是在连接位点引入 C 端活化肽段和 N 端半胱氨酸肽段，在水相中先形成硫酯中间体，
随后经分子内 \textit{S}$\rightarrow$\textit{N} 重排生成天然肽键。为避免传统硫酯片段在纯化过程中的不稳定性，
本实验采用“酰肼 $\rightarrow$ 酰基叠氮 $\rightarrow$ 硫酯中间体”的改进策略，将 H3 N 端酰肼六肽与 H3 C 端蛋白片段拼接，
构建组蛋白 H3 的合成模型\cite{zhang2024,collins2023}。

本实验综合运用MW-SPPS、重组蛋白纯化（包涵体提取、His-tag亲和层析、SDS-PAGE、Bradford定量、SUMO酶切、TCA沉淀）及NCL连接技术，完成组蛋白H3的化学合成。实验分为三个模块：模块I合成H3 N端酰肼六肽（H-ARTKQT-NHNH₂），模块II纯化获得N端为Cys的H3 C端蛋白，模块III通过NCL连接两个片段获得全长H3。

\section{实验部分}
\vspace{-0.5em}

\subsection{仪器与试剂}

模块 I 主要使用微波合成仪（CEM Discovery SP）、
Agilent 1100 高效液相色谱、Autoflex Max MALDI-TOF 质谱仪、半制备色谱及冻干设备；
模块 II 主要使用超声破碎仪、Ni-TED 亲和纯化柱、电泳仪、PCR 仪、酶标仪和超滤离心管等。主要试剂包括
Fmoc 保护氨基酸、Fmoc-NHNH-CTC 树脂、Oxyma Pure、DIC、TFA、DHB、GuHCl、尿素、Bradford 试剂和 Ulp1 蛋白酶等。

\subsection{实验方法}

\textbf{模块 I：H3 N 端酰肼六肽的合成与表征。}
以 Fmoc-NHNH-CTC 树脂为固相载体，在 DMF/DCM 混合溶剂中完成树脂溶胀后，
按照 Thr--Gln--Lys--Thr--Arg--Ala 的顺序进行多肽链延伸。脱保护条件采用 20\% 哌啶/DMF，90~$^{\circ}$C 微波加热约 70~s；
偶联反应采用 DIC/Oxyma/DIPEA 体系，在 50~$^{\circ}$C 下反应 5--10~min，其中 Arg 残基采用重复偶联以提高效率。
多肽合成完成后，使用 TFA/TIS/H$_2$O（95/2.5/2.5, v/v/v）于 38~$^{\circ}$C 条件下切割 35~min，
再经冰乙醚沉淀得到粗肽。色谱表征采用 Agilent tc-C18(2) 色谱柱（4.6$\times$150~mm，5~$\mu$m），
流动相 A 为 0.1\% TFA/水，B 为 0.1\% TFA/乙腈，0--2~min 维持 5\% B，2--30~min 由 5\% B 线性升至 100\% B，
流速 0.8~mL$\cdot$min$^{-1}$，检测波长 214~nm。质谱采用正离子模式 MALDI-TOF 测定分子量。

\clearpage

\textbf{模块 II：H3 C 端蛋白的纯化与表征。}
表达有 His-SUMO-H3 C 端片段的菌体先在冰水浴中加入 15--20~mL 裂解缓冲液超声破碎，
按“超声 5~s、暂停 10~s、总时长约 15~min”的节奏完成细胞裂解；随后在 4~$^{\circ}$C、11000~r$\cdot$min$^{-1}$ 条件下离心 15~min 收集包涵体。
得到的沉淀先用含 1\% Triton X-100 的裂解缓冲液洗涤，再用 IB washing buffer 重复洗涤一次，以尽可能去除膜碎片和可溶性杂蛋白。
纯化后的包涵体用含 6~mol$\cdot$L$^{-1}$ GuHCl 的 IB extraction buffer 溶解，离心去除不溶残渣后，沿纯化柱内壁上样至 1~mL Ni-TED 树脂。
树脂依次用 6~mL IB extraction buffer 和 3~mL Urea washing buffer 充分洗涤，最后使用含 250~mmol$\cdot$L$^{-1}$ 咪唑的 Urea elution buffer 洗脱含 His-tag 的目标蛋白，
按每管 0.5~mL 收集 10 管流份，并用 Bradford 试剂在 parafilm 膜上进行快速定性筛查。

对含蛋白的流份，每管取 5~$\mu$L 与 10~$\mu$L 2$\times$上样缓冲液混合，经 95~$^{\circ}$C 加热 3~min 后上样至 12\% SDS-PAGE 胶，
在 180~V 下电泳约 40--45~min，染色与脱色后判断目标蛋白主要分布于第 2--6 管。将这些高纯度流份合并后转入 10~kDa 超滤管，在室温下浓缩至约 1~mL，
再逐滴加入 5~mL Refolding buffer 中完成复性，并继续超滤至约 0.5~mL。Bradford 法浓度测定中，
以 2.0~mg$\cdot$mL$^{-1}$ BSA 储备液配制 0.125、0.25、0.50、0.75 和 1.00~mg$\cdot$mL$^{-1}$ 五个浓度梯度，
每孔加入 5~$\mu$L 标准液、Refolding buffer 空白或待测样品，再加入 245~$\mu$L Bradford 试剂，在 595~nm 下读取吸光度。
最后按蛋白:Ulp1~=~100:3 的摩尔比加入 Ulp1 蛋白酶，在 4~$^{\circ}$C 或室温条件下酶切 6--12~h，并用 SDS-PAGE 比较酶切效果。

\textbf{模块 III：NCL 技术路线说明。}
模块 III 以模块 I 获得的 H3 N 端酰肼六肽和模块 II 获得的 H3 C 端蛋白为底物。
实验时先将约 425~$\mu$g 冻干六肽溶于 115~$\mu$L Acidic Ligation Buffer，并在 $-12$~$^{\circ}$C 冷阱中预冷 5~min；
随后缓慢加入 8.6~$\mu$L 0.2~mol$\cdot$L$^{-1}$ NaNO$_2$，在低温下反应 20~min，使酰肼转化为酰基叠氮中间体。
与此同时，用 125~$\mu$L 0.3~mol$\cdot$L$^{-1}$ MPAA 溶液溶解约 0.1~mg H3 C 端蛋白，并滴加到上述肽段溶液中，
再用 2~mol$\cdot$L$^{-1}$ NaOH 将体系调节至 pH~6.8，以提高 Cys 侧链亲核性并促进后续的硫酯交换与酰基转移。
反应在室温下继续搅拌约 6~h；监测时取 20~$\mu$L 反应液，加入 1.35~$\mu$L 0.2~mol$\cdot$L$^{-1}$ TCEP 还原 30~min，
再补加 20~$\mu$L 色谱 A 相，经高速离心后使用与模块 I 相同的 HPLC 条件分析，并收集 15--30~min 的主要峰进行 MALDI-ToF-MS 表征。

HPLC 结果见图~\ref{fig:hplc_crude}--\ref{fig:hplc_overlay}。6-2 粗肽样品在 3.21~min 附近给出显著主峰，说明目标六肽已经成功生成。该主峰前后仍伴随多个低保留时间和中高保留时间杂峰，

从 6-2 的色谱形态看，目标主峰之前的弱峰可归因于更高极性的截短肽或二酮哌嗪副产物，而 5--15~min 区间的宽峰与肩峰则更可能来自保护基未完全脱除的副产物、
TFA 加合物或少量树脂裂解片段。主峰相对纯肽标准后移约 0.10~min，也与粗肽中残留较多 TFA 盐、离子对作用增强疏水性这一解释相一致。
因此，对 6-2 粗肽更合理的结论不是简单给出某个“主峰面积百分比”，而是确认其已经含有目标六肽、但杂质组成仍较复杂；后续若要进一步提高回收率和纯度，
优先应从 Arg 双偶联、切割条件优化及后处理洗涤充分性三方面入手。

与粗肽色谱中“主峰附近伴随杂峰”的现象相互印证。

综上，模块 I 的实验结果说明：MW-SPPS 已成功获得目标酰肼六肽，HPLC 和质谱都支持目标产物的存在，
但粗肽纯度仍受副反应和后处理因素影响，因而色谱纯化和后续结构鉴定仍是不可缺少的步骤。

\section{结论}
\vspace{-0.5em}

本实验以天然化学连接法合成组蛋白 H3 为目标，围绕“多肽合成--蛋白纯化--后续连接”三阶段技术路线展开，
完成了 H3 N 端酰肼六肽与 H3 C 端蛋白片段的关键表征。主要结论如下：

模块 I 中，6-2 粗肽样品的表观产率明显高于理论值，说明粗肽中同时含有 TFA 盐、保护基残留和副产物；
HPLC 显示 6-2 粗肽在 3.21~min 附近出现目标主峰，并与纯肽标准的 3.11~min 主峰相互对应。
MALDI 质谱进一步确认 $m/z$ 718.358 的主峰与目标六肽理论分子量吻合，证明 H3 N 端酰肼六肽已成功合成。
另外，在合成过程中出现了意料之外的情况：切割后体系中有绿色的物质（图~\ref{fig:photo_cleavage}），在加入乙醚后迅速地消失，而重新加入 TFA 后又恢复；猜测这可能与氯代三苯甲基碳正离子相关。

模块 II 中，经包涵体提取、Ni-TED 亲和纯化和超滤复性后，His-SUMO-H3 C 端蛋白能够以较高纯度和较高浓度回收；从2.5 g湿菌体中提取包涵体，经两次洗涤后获得0.2615 g包涵体湿重，洗涤去除率约6.5\%。Ni-TED亲和纯化后，SDS-PAGE显示融合蛋白（His-SUMO-H3）在27 kDa处呈现单一主带，洗脱效果良好。
依据 Bradford 标准曲线，最终蛋白浓度约为 $(3.57\pm0.15)$~mg$\cdot$mL$^{-1}$。Ulp1 酶切后出现与 SUMO 标签和 H3 C 端片段相符的新条带，且 4~$^{\circ}$C 条件优于室温。

本实验通过 MW-SPPS、HPLC、MALDI-TOF、SDS-PAGE 与蛋白定量等方法，成功地实现了组蛋白 H3 的化学合成与生化表征。
